A sequence \(\{ a_n\} \) converges to \(L ∈ \mathbb {R}\) if for all \(\varepsilon {\gt} 0\) there exists \(N ∈ ℕ\) such that for all \(n ≥ N\), \(|a_n - L| {\lt} \varepsilon \). We say \(\{ a_n\} \) converges if there exists \(L ∈ \mathbb {R}\) such that \(\{ a_n\} \) converges to \(L\).

Definition 3 Bounded Above
✓
#

A set \(S \subseteq \mathbb {R}\) is bounded above if there exists an \(M \in \mathbb {R}\) such that for all \(s \in S\), \(s \le M\). We say that \(M\) is an upper bound for \(S\).

Let \(S \subseteq \mathbb {R}\) be non-empty. We say that \(y \in \mathbb {R}\) is the supremum of \(S\) if

(i) \(y\) is an upper bound for \(S\)
(ii) for any upper bound \(z\) of \(S\), \(y \le z\)

A function \(f : \mathbb {R} → \mathbb {R}\) is continuous if for all \(x \in \mathbb {R}\) and for all \(\varepsilon {\gt} 0\), there exists a \(\delta {\gt} 0\) such that for all \(y ∈ \mathbb {R}\) with \(|x - y| {\lt} \delta \), \(|f(x) - f(y)| {\lt} \varepsilon \).

Theorem 10 Intermediate Value Theorem
✓
#

Uses

  • Theorem 9
  • Definition 5
  • Theorem 8

Lean declarations

  • intermediate_value

Let \(f : \mathbb {R} → \mathbb {R}\) be continuous and let \(a {\lt} b ∈ \mathbb {R}\). Suppose that \(f(a) {\lt} f(b)\) and let \(y \in [f(a), f(b)]\). Then there exists a \(c ∈ [a, b]\) such that \(f(c) = y\).

Proof ▶

Sketch:
Let \(K = \{ x \in [a, b] | f(x) \le y\} \). Then \(K\) is non-empty and bounded above by \(b\), so it has a supremum \(c \in \mathbb {R}\). If \(f(c) {\lt} y\), then by continuity of \(f\), there exists a \(\delta {\gt} 0\) such that for all \(x ∈ (c - \delta , c + \delta )\), \(f(x) {\lt} y\). Choose \(x = c + \delta /2\).Then \(x ∈ K\) and \(x {\gt} c\), so \(c\) is not an upper bound for \(K\). If \(f(c) {\gt} y\), then by continuity of \(f\), there exists a \(\delta {\gt} 0\) such that for all \(x ∈ (c - \delta , c + \delta )\), \(f(x) {\gt} y\). Choose \(x = c - \delta /2\). Then \(x\) is an upper bound for \(K\) and \(x {\lt} c\), so \(c\) is not the supremum of \(K\). Thus, \(f(c) = y\).

0.3 Modular Forms

Definition 11 Modular Form
✓
#

Lean declarations

  • ModularForm

In lean, A modular form of weight \(k \in \mathbb {N}\) is a function \(f : \mathbb {C} → \mathbb {C}\) such that :

(1) \(f\) is holomorphic on \(\mathbb {H}\)
(2) For all \(z \in \mathbb {H}, f(z) = f(z + 1)\)
(3) For all \(z \in \mathbb {H}, f(z) = z^{-k} f(-1/z)\)
(4) \(f\) is bounded as Re\((z) \to \infty \)
